ALARM SOUND FILE
================

Place your alarm.wav file in this directory.

You can download a free short alarm sound from:
  https://freesound.org/search/?q=alarm+beep

Or generate a simple beep using Python:

    python generate_alarm.py

The generate_alarm.py script is included in this project and creates
a simple 1-second beep alarm.wav without needing any downloads.
